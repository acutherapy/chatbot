🧠 完整 AI 系统配置步骤

  1. 配置环境变量：在 Vercel 项目设置中添加以下环境变量：
    OPENAI_API_KEY=your_openai_api_key_here META_PAGE_ACCESS_TOKEN=your_meta_page_access_token_here META_VERIFY_TOKEN=your_meta_verify_token_here META_APP_SECRET=your_meta_app_secret_here
  2. 获取 OpenAI API Key：访问 OpenAI Platform 创建 API Key
  3. 配置 Meta 平台：在 Meta for Developers 创建应用
  4. 重新部署：配置完成后重新部署到 Vercel
