🤖 聊天机器人网站集成指南

方法 1: 使用 JavaScript SDK（推荐）

这是最简单和最灵活的方法，支持自定义配置。

<!-- 在你的网站 HTML 中添加 --> <script src="https://y-dkhr1576t-david-cais-projects-ae09bbdf.vercel.app/js/chatbot-sdk.js"></script> <script> // 初始化聊天机器人 const chatbot = new ChatbotSDK({ position: 'bottom-right', // 位置: bottom-right, bottom-left, top-right, top-left theme: 'default', // 主题: default, dark, light autoOpen: false, // 是否自动打开 delay: 3000 // 自动打开延迟（毫秒） }); </script>

方法 2: 使用 iframe 嵌入

直接嵌入聊天界面，适合需要完全控制样式的场景。

<!-- 浮动聊天窗口 --> <iframe src="https://y-p3iw34mat-david-cais-projects-ae09bbdf.vercel.app/embed.html" width="350" height="500" frameborder="0" style="position: fixed; bottom: 20px; right: 20px; z-index: 1000; border-radius: 10px; box-shadow: 0 4px 20px rgba(0,0,0,0.15);"> </iframe>

方法 3: 聊天按钮 + 弹窗

创建自定义的聊天按钮和弹窗，完全控制用户体验。

<!-- 聊天按钮 --> <button id="chatToggle" style=" position: fixed; bottom: 20px; right: 20px; width: 60px; height: 60px; border-radius: 50%; background: #4F46E5; color: white; border: none; cursor: pointer; z-index: 1000; ">💬</button> <!-- 聊天窗口 --> <div id="chatWindow" style=" position: fixed; bottom: 90px; right: 20px; width: 400px; height: 600px; border-radius: 10px; box-shadow: 0 4px 20px rgba(0,0,0,0.15); display: none; z-index: 1000; "> <iframe src="https://y-dkhr1576t-david-cais-projects-ae09bbdf.vercel.app" width="100%" height="100%" frameborder="0" style="border-radius: 10px;"> </iframe> </div> <script> document.getElementById('chatToggle').addEventListener('click', function() { const chatWindow = document.getElementById('chatWindow'); chatWindow.style.display = chatWindow.style.display === 'none' ? 'block' : 'none'; }); </script>

方法 4: 使用配置脚本

通过配置脚本自动初始化，适合需要统一管理的场景。

<!-- 配置脚本 --> <script type="application/json" data-chatbot-config> { "position": "bottom-right", "theme": "default", "autoOpen": true, "delay": 5000 } </script> <script src="https://y-dkhr1576t-david-cais-projects-ae09bbdf.vercel.app/js/chatbot-sdk.js"></script>
💡 提示：
  • 确保你的网站支持 HTTPS（聊天机器人需要安全连接）
  • 可以根据需要调整位置、大小和样式
  • 建议在移动端使用较小的尺寸
  • 可以通过 API 发送消息到聊天机器人
